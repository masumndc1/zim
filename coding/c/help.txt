... explanation:

extern char **environ;

-- extern: Tells the compiler that the variable environ is defined somewhere
-- else (managed by the OS/runtime library), so it should not allocate new memory for it.
-- It just looks up the existing one during linking.

-- char **: A pointer to a pointer of characters. Think of it as an array of strings
-- (char *environ[]). Each element points to a null-terminated string formatted as KEY=value
-- (e.g., "USER=alice").environ: The standard POSIX-defined variable name for this environment list.
