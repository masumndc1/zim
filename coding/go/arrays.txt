[]string: The exact data type of the slice components
 (e.g., strings, integers, custom structs).
length: The initial number of elements populated with safe "zero-values"
 (e.g., "" for strings, 0 for ints) that you can read or overwrite immediately.
capacity (Optional): The total number of elements the underlying memory array
 can hold before Go is forced to resize it.
