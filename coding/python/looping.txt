

... looping in python
... Here are the various ways to loop in Python:

... 1. The for Loop (The Workhorse)
... Python’s for loop is actually an "iterator."
... It’s used to go through items in a sequence
... (like a list, tuple, or string).

services = ["nova", "neutron", "keystone"]
for service in services:
    print(f"Checking status of: {service}")

... 2. The range() Loop
... When you need to repeat an action a specific number of times or need an index.

# range(start, stop, step)
for i in range(1, 6, 2):
    print(f"Retry attempt {i}")
    # Prints 1, 3, 5

... 3. The while Loop
... Runs as long as a condition is True.
... This is common in OpenStack scripts when waiting for a VM to
...  move from BUILD to ACTIVE status.

status = "BUILDING"
attempts = 0

while status != "ACTIVE" and attempts < 5:
    print("Waiting for instance...")
    attempts += 1

... 4. List Comprehensions (The "Pythonic" Way)
... A concise, one-line loop used to create new lists.
... Developers love these for filtering data.

# Get a list of only 'high-priority' flavor IDs
flavors = [
    {"id": 1, "ram": 512},
    {"id": 2, "ram": 4096}
  ]

high_ram = [f["id"] for f in flavors if f["ram"] > 1024]

... 5. Dictionary Loops
... Useful for processing metadata or configuration files.

config = {
  "region": "us-east",
  "debug": True
}

for key, value in config.items():
    print(f"Setting {key} to {value}")

... Loop Control Statements
break: Exit the loop immediately.
continue: Skip the rest of the current block and move to the next iteration.
