Created Tuesday 26 November 2013

installing gns3 in debian
# apt-get install gns3 qemu
# updatedb
# locate qemuwraper.py
# locate qemu
# locate qemu-img

if qemuwraper.py not found then download gns3-0.8.tar.gz and then untar 
it and then on extract folder qemuwrapper.py can be found.
